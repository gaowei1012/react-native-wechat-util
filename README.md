# react-native-wechat-util

reactnative for wechat utils

## Installation

```sh
npm install react-native-wechat-util
```

## Usage

```js
import WechatUtil from "react-native-wechat-util";

// ...

const result = await WechatUtil.multiply(3, 7);
```
