# react-native-wechat-util

react-native-wechat-util

> react-native版本要求 >= 0.60, 目前此项只集成了`android`项目功能。

> 安装
```shell
  npm install react-native-wechat-util
```

#### 已完成功能

- [x] isWxInstalled 检查本机是否安装微信客户端
- [x] registerApp('appId') 注册appId
- [x] openWxApp 打开微信客户端
- [x] getAppVersion 获取微信版本号
- [x] sendAuthRequest(scope, state) 微信授权登录

#### 开发中功能

- [ ] pay 微信支付功能
- [ ] 分享功能
